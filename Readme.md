##Side Hustle
To start the server:
1. Update dependencies using: <code>npm install</code>
2. Run Migrations using: <code>sequelize db:migrate</code>
3. Run Seeding using: <code>sequelize db:seed:all</code>
4. Start server using <code>npm run dev</code>
